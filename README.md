# wiener-filter-noise-suppression
A MATLAB-based noise suppression algorithm that applies a Wiener filter to speech audio.
Supports automatic noise profile estimation, with configurable gain flooring and noise scaling.

Noisy audio is sourced from the NOIZEUS speech corpus. Available to researchers courtesy of Dr. Philip Loizou et al. at UT Dallas.
Link: https://ecs.utdallas.edu/loizou/speech/noizeus/

The "street_10db" and "street_5db folders" contain audio files from the NOIZEUS dataset with signal-to-noise ratios (SNRs) of 10dB and 5dB, respectively.
